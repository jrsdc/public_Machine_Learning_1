# Machine Learning & Data Mining Coursework

Predictive modelling project completed as part of an undergraduate Machine Learning and Data Mining module. The project applies supervised learning techniques to a loan application dataset to address two real-world problems in consumer lending.

## Project Overview

The coursework addresses two research questions for a lending institution:

1. **Case Study A (Classification)** — Can machine learning assist bankers and finance analysts in predicting which clients should be approved for a loan?
2. **Case Study B (Regression)** — For loan-approved clients, can machine learning offer a reliable estimate of the maximum lending value?

## Methodology

The project follows the CRISP-DM methodology:

- Data understanding and exploratory analysis
- Data preparation — handling missing values, outliers, and categorical encoding
- Modelling — multiple algorithms trained and evaluated
- Evaluation — performance metrics aligned to business success criteria
- Hyperparameter tuning and ensemble methods

## Models Built

**Classification**
- Logistic Regression
- Naive Bayes
- K-Nearest Neighbours
- Voting Ensemble Classifier (soft voting)

**Regression**
- Decision Tree Regressor — fully grown
- Decision Tree Regressor — pruned (max_depth=4)
